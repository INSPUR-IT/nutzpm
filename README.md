# nutzpm
A j2ee project base on nutz framework,includes nutz framework、bui web framework、shiro security framework
- 基于Nutz框架实现的基于MySQL数据库的权限管理系统，包括基本的组织机构、用户、角色、权限管理。

### 框架使用以下技术实现：
#### Nutz、Nutz-Web
#### BUI前端框架
#### Shiro安全框架
#### Freemarker模板引擎
#### Druid数据库连接池组件
#### Quartz定时任务组件
